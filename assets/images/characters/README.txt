# Image Assets Folder

## Character Images
Place your character sprite images in this folder.

**Naming convention:**
- `recon_idle.png` - Recon character
- `heavy_idle.png` - Heavy character  
- `tech_idle.png` - Tech character
- `medic_idle.png` - Medic character
- `stealth_idle.png` - Stealth character

**Recommended size:** 32x48 pixels (or larger, will be scaled)
**Format:** PNG with transparency

## Creating Pixel Art

You can use:
- Aseprite (recommended)
- Piskel (free online)
- GraphicsGale
- Photoshop/GIMP

**Tips:**
- Use a small palette (4-8 colors)
- Start with a simple silhouette
- Add details last
- Save as PNG with transparency
